Chapter 6: Programming Languages

Term Definition
Machine language The language used by computers that consists of binary code, representing instructions that a computer can execute.
Debugging The process of finding and resolving errors in software.
Assembler A program that translates assembly language into machine language.
Assembly language A low-level programming language that is specific to a particular computer architecture and can be directly converted to machine language.
Translator A program that converts code from one language to another.
Compiler A program that translates high-level programming language into machine language.
Interpreter A program that reads code and executes it directly.
Cross-platform software Software that can run on multiple operating systems.
Natural language A human language used for communication.
Formal language A language that is designed for use in specific applications, such as programming.
Machine independence The ability of software to run on different types of hardware without modification.
High-level programming language A programming language that is easier for humans to read and write than low-level languages.
Object-oriented programming (OOP) A programming paradigm that focuses on the use of objects to represent data and methods to manipulate that data.
Scripting language A programming language used to automate frequently executed tasks.
Data types:
 Integer A data type that represents whole numbers.
 Float A data type that represents decimal numbers.
 String A data type that represents a sequence of characters.
Primitive data type A basic data type that is not derived from other data types.
Data structure A way of organizing and storing data in a computer so that it can be accessed and used efficiently.
*Python collection data structures:
 Lists A list is a collection that is ordered and changeable. In Python, lists are created using square brackets [] with comma-separated items inside. Lists can contain items of different data types.
 Tuple A tuple is a collection that is ordered and unchangeable. In Python, tuples are created using parentheses () with comma-separated items inside. Tuples can contain items of different data types.
 Set A set is a collection that is unordered and unindexed. In Python, sets are created using curly braces {} or the set() constructor function with comma-separated items inside. Sets can only contain unique items.
 Dictionary A dictionary is a collection that is unordered, changeable, and indexed. In Python, dictionaries are created using curly braces {} or the dict() constructor function with key-value pairs separated by colons inside. Dictionaries can contain items of different data types and are accessed using keys rather than indexes.
Function A set of instructions that performs a specific task or set of tasks within a program, which can be called and reused throughout the code.
Local variable A variable that is declared within a specific function or block of code and can only be accessed within that function or block.
Global variable A variable that is declared outside of any specific function or block of code and can be accessed and modified throughout the program.
Source program The human-readable version of a computer program, typically written in a high-level programming language.
Object program The machine-readable version of a computer program, created by compiling or interpreting the source program.
