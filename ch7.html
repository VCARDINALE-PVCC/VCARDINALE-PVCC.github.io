Briefly describe or define:

Term Definition
Computer network A group of interconnected computers and other devices that communicate and exchange data with each other. It can be either wired or wireless and can span a small area such as a local area network (LAN) or a large area such as a wide area network (WAN).
Analog device A type of electronic device that works with continuous physical quantities such as voltage, current, and sound waves. Examples of analog devices include analog clocks, vinyl records, and old-style telephones.
Digital device A type of electronic device that works with discrete values or signals such as binary digits (bits). Examples of digital devices include digital cameras, digital watches, and smartphones.
Modem A device that connects a computer or other digital device to the internet or another computer network. It converts digital signals from a computer into analog signals that can be transmitted over telephone or cable lines and vice versa.
Bandwidth The maximum amount of data that can be transmitted over a network in a given amount of time. It is usually measured in bits per second (bps) or megabits per second (Mbps).
Ethernet A type of wired computer networking technology that uses a set of standards to transmit data over a network. It is one of the most widely used network technologies today.
WLAN Wireless Local Area Network A type of computer network that uses wireless technology to connect devices within a limited geographic area such as a home, office, or campus.
IEEE 802.11 wireless network standard A set of protocols that govern the use of wireless technology in computer networks. It is commonly known as Wi-Fi and is used in most wireless networks today.
MAN Metropolitan Area Network A type of computer network that spans a metropolitan area such as a city or town.
Internet of Things A network of physical devices, vehicles, buildings, and other objects that are connected to the internet and can exchange data with each other.
LAN Local Area Network A type of computer network that spans a small area such as a single building or campus.
Repeater A device that receives a signal and retransmits it at a higher power or to a wider area. It is used to extend the range of a network or to improve the quality of a signal.
Bridge A device that connects two or more computer networks together and forwards data between them. It operates at the data link layer of the OSI model.
WAN Wide Area Network A type of computer network that spans a large geographic area such as a state, country, or continent.
Packet and packet switching A packet is a unit of data that is transmitted over a network. Packet switching is a method of transmitting data in which packets are sent individually and can take different paths to their destination.
ISP Internet Service Provider A company that provides internet access to customers.
Firewall A security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
Protocol A set of rules that govern the way data is transmitted over a network. It defines how data is formatted, transmitted, received, and processed.
Cloud computing A type of computing that relies on remote servers to store, manage, and process data. It allows users to access computing resources on demand over the internet.
Net neutrality The principle that all internet traffic should be treated equally by ISPs, without any discrimination or preference given to certain websites, services, or users. It aims to promote a free and open internet, where all users have equal access to information and services.

Briefly describe the five layers of TCP/IP (the Internet protocol hierarchy):

  1. Physical layer: This layer defines the physical aspects of data transmission, such as the type of cables, connectors, and signal voltages used to transmit data over a network.
  2. Data link layer: This layer is responsible for the reliable transmission of data between two devices on the same network. It includes protocols such as Ethernet, Wi-Fi, and Bluetooth.
  3. Network layer: This layer handles the routing of data between different networks. It includes protocols such as IP (Internet Protocol) and ICMP (Internet Control Message Protocol).
  4. Transport layer: This layer is responsible for the reliable delivery of data between applications running on different devices. It includes protocols such as TCP (Transmission Control Protocol) and UDP (User Datagram Protocol).
  5. Application layer: This layer provides the interface between the network and the applications running on the devices. It includes protocols such as HTTP (Hypertext Transfer Protocol), FTP (File Transfer Protocol), and SMTP (Simple Mail Transfer Protocol).

List and briefly describe at least 3 network services and benefits:

  1. Email Service: Email service is a network service that enables users to send and receive electronic messages over a network. Email is a fast and reliable way to communicate with others, regardless of their location or time zone. It allows for easy sharing of files and attachments, and it is accessible from multiple devices such as computers, smartphones, and tablets.
  2. File Sharing Service: File sharing service is a network service that allows users to share files with others over a network. This service is commonly used for collaborative work, allowing multiple users to access and edit the same document at the same time. It is also useful for sharing large files, such as videos or high-resolution images, that would be difficult to share via email.
  3. Remote Access Service: Remote access service is a network service that enables users to access a network or computer from a remote location. This service is useful for telecommuting, allowing employees to work from home or on the go. It also provides IT professionals with remote access to servers and other network resources, allowing for quick and efficient troubleshooting and maintenance.
