All About GitHub

  • Q. What is a version control system?
  • A. tracks the history of changes as people and teams collaborate on projects together.
  • Q. What is Git?
  • A. A free and opensource distributed version control system
  • Q. What is GitHub?
  • A. A web-based interface that uses Git to let the public make seporate changes to web pages at the same time
  • Q. What is the difference between Git and GitHub?
  • A. While Git is a opensourced software that is installed locally on a system--GitHub is a service that implements Git on the internet
  • Q. Who started GitHub and how was it started?
  • A. GitHub started in 2008 by Chris Wanstrath, P. J. Hyett, Tom Preston-Werner, and Scott Chaconusing using Ruby on Rails.
  • Q. What company owns it now?
  • A. Microsoft
  • Q. How much does a GitHub account cost?
  • A. it is free for personal accounts, however it is $44 a year per account for a team of developers
  • Q. What is Octocat?
  • A. it is a half cat half octopus mascot for GitHub

Git/GitHub terms:

  • Repository: a folder in which all files and their version histories are stored.
  • Commit: a saved record of a change made to a file within the repo.
  • Fork: a new repository that shares code and visibility settings with the original “upstream” repository.
  • Push: updates the remote repository with any commits made locally to a branch.
  • Pull Requests: the way to ask for changes made to a branch to be merged into another branch that also allows for multiple users to see, discuss and review work being done.
  • Workflows: a configurable automated process that will run one or more jobs.
  • Issues: how work is tracked when using git. Issues allow users to report new tasks and content fixes, as well as allows users to track progress on a project board from beginning to end of a specific project.
  • Raw Button: opens the file in a raw form, meaning that any HTML formatting disappears.
  • Blame Button: describes the last modification to each line of a file, which generally displays the revision, author and time.
