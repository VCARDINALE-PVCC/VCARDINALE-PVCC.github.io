Operating System Glossary

Term Definition
Operating system A software program that manages and controls the hardware and software resources of a computer system.
Job queue A sequence of jobs waiting to be processed by a computer system or a specific device.
FIFO First In, First Out. A data structure in which the first item added is the first item to be removed.
Interactive processing A mode of operation in which the computer interacts with the user in real-time, allowing the user to give immediate input and receive immediate feedback.
Time-sharing A technique used by operating systems to allow multiple users to share a computer's resources simultaneously, by allocating CPU time in small, time-sliced increments.
Multiprogramming A technique used by operating systems to allow multiple programs to run on a computer at the same time.
Application software Software designed to perform specific tasks or functions for the user, such as word processing, accounting, or gaming.
Systems software Software designed to provide basic functionality and services for the computer system, such as the operating system, device drivers, and utilities.
Utility software Software designed to perform specific maintenance or diagnostic tasks on the computer system, such as antivirus software or disk cleanup tools.
Graphical user interface A visual interface that allows users to interact with a computer system using icons, buttons, and other graphical elements.
Shell A program that provides a command-line interface for users to interact with the operating system.
Kernel The core of an operating system that provides basic services such as memory management, process management, and input/output (I/O) management.
Device driver Software that allows the operating system to communicate with and control hardware devices attached to the computer system.
File manager Software that manages the allocation and deallocation of memory resources in a computer system.
Virtual memory A technique used by operating systems to simulate more memory than is physically available by using disk space as additional memory.
Boot strapping The process of loading and initializing the operating system on a computer system when it is first powered on or restarted.
ROM Read-Only Memory. A type of computer memory that can only be read, not written to, and is often used for firmware or system-level software.
Deadlock A situation in which two or more processes are unable to continue executing because each is waiting for the other to release a resource.
Spooling Simultaneous Peripheral Operations On-Line. A technique used by operating systems to manage the input and output of data between devices and the computer system.
Super user (or administrator) A user account with elevated privileges that allows the user to perform administrative tasks on a computer system.
Multi-factor authentication A security process that requires users to provide two or more forms of authentication, such as a password and a fingerprint, to access a computer system or application.
