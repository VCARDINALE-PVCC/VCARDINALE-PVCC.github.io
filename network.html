Chapter 4 Terms

Network Classifications:

  • LAN (Local Area Network) - A network that connects devices within a small geographical area, such as a home, office, or school.
  • MAN (Metropolitan Area Network) - A network that connects devices within a larger geographical area, such as a city or town.
  • WAN (Wide Area Network) - A network that connects devices across a large geographical area, such as a country or the world.

Network Models:

  • Bus Network - A network topology in which all devices are connected to a single cable or "bus."
  • Star Network - A network topology in which all devices are connected to a central hub or switch.

Protocols:

  • CSMA/CD Ethernet - A protocol used for wired LANs that allows multiple devices to share the same communication medium.
  • CSMA/CA Wifi - A protocol used for wireless LANs that helps prevent collisions between devices attempting to transmit data.

Combining Networks:

  • Repeater - A device that receives a signal and amplifies it, allowing it to travel further.
  • Bridge - A device that connects two LANs together, allowing them to communicate with each other.
  • Switch - A device that connects multiple devices on a LAN, directing traffic between them.
  • Router - A device that connects two or more networks together, directing traffic between them.
  • Gateway - A device that connects two different types of networks together, such as a LAN and a WAN.

Client/Server Model:

  • P2P (Peer-to-Peer)- A network model in which all devices are equal and can act as both clients and servers, sharing resources and services with each other.
  • Distributed System- A network model in which multiple devices work together to accomplish a common goal, often by dividing the workload among them.
  • Cloud Computing- A model in which computing resources and services are delivered over the internet, typically on a pay-per-use basis.

Internet Addressing:

  • Tier 1 - A global network that connect directly to other Tier 1 networks.
  • Tier 2 - Tier 2 ISPs connect to multiple Tier 1 networks and other Tier 2 networks.
  • Tier 3 - A regional network that connect to one or more Tier 2 networks.

IP Address - A unique numerical identifier assigned to every device connected to the internet.

  • ICANN (Internet Corporation for Assigned Names and Numbers) - An organization responsible for managing domain names and IP address allocations.
  • DNS (Domain Name System) - A system that translates domain names (e.g. www.example.com) into IP addresses.
  • TLD (Top-Level Domain) - The last part of a domain name, such as .com, .org, or .net.
  • Country-code TLDs - TLDs that correspond to specific countries or territories, such as .uk for the United Kingdom or .ca for Canada.

Internet Protocols:

  • FTP (File Transfer Protocol) - A protocol used for transferring files between computers.
  • HTTP (Hypertext Transfer Protocol) - A protocol used for transmitting web pages and other content over the internet.
  • SMTP (Simple Mail Transfer Protocol) - A protocol used for sending and receiving email.

World Wide Web:

  • A system of interconnected hypertext documents accessed via the internet.
  • URL (Uniform Resource Locator) - A string of characters that specifies the location of a resource on the web.
  • Client-Side Activities - Actions performed by a user's web browser, such as displaying a web page or running JavaScript code.
  • Server-Side Activities - Actions performed by a web server, such as processing form data or generating dynamic web content.

Internet Four-Layer Software:

  • Application Layer - The layer responsible for providing services to users, such as email or web browsing.
  • Transport Layer - The layer responsible for ensuring reliable communication between devices.
  • Network Layer - This layer is responsible for routing data between devices on different networks. The Internet Protocol (IP) is the primary protocol used at this layer.
  • Link Layer - This layer is responsible for managing communication between devices on the same network. Ethernet is an example of a protocol that operates at this layer.

Internet is:

  • Packet-switching network - A network where data is divided into small packets for transmission.
  • Fault tolerant - A system's ability to continue functioning even when there is a fault or failure.
  • Redundant - The use of additional components or resources to provide backup in case of a failure.

Cybersecurity

  • Malware - Malicious software that is designed to harm a computer or steal data.
  • Virus - A type of malware that can replicate itself and spread to other devices.
  • Worm - A self-replicating program that can spread quickly across a network.
  • Trojan Horse - A type of malware that disguises itself as legitimate software.
  • Spyware - Software that is designed to collect information from a computer without the user's knowledge.
  • Phishing - A type of social engineering attack that is used to trick users into revealing sensitive information.
  • DDoS Attack - A type of attack where a network or website is flooded with traffic to make it unavailable.

Protection

  • Firewall - A software or hardware device that is used to block unauthorized access to a network.
  • Spam Filter - A program that is used to identify and remove unwanted email messages.
  • Proxy Server - A server that acts as an intermediary between a device and the internet, providing an extra layer of security and privacy.
  • Antivirus Software - Software that is designed to detect and remove malware from a computer.
  • Cryptography - The practice of using codes and ciphers to protect sensitive information.
  • Public Key Encryption - A method of encrypting data using two keys, one for encryption and one for decryption, that are publicly shared and kept private, respectively.
