Three misconceptions about Computer Science:

  1. Computer Science is the study of computers.
  2. Computer science is the study of how to write computer programs.
  3. Computer science is the study of the uses and applications of computers and software.

Breifly define:

1. Computer Science
The study of algorithms
2. algorithm
A step-by-step method for accomplishing some task
3. sequential operation
Performs out a single well-defined task
4. conditional operation
Instructions that ask a question then alter the next operation accordingly
5. iterative operation
An algorithm that instructs the program to loop a set of instructions
6. stored program computer
A machine that uses internal wireing and stores program instructions instead of machine data
7. programmer
A person who writes code in high-level programming languages
8. integrated circuit
A set of circuts etched on one small semiconductive piece of material

Briefly describe these computer generations

1. First generation computer
1950-1957:
First commercial computers
First symbolic programming languages
Use of binary arithmatic, vaccum tubes for storage
Punched card input/output
2. Second generation computer
1957-1965
Transistors and core memories
First disks for mass storage
Size reduction, increased reliability, lower costs
First high-level programming languages
First operating systems
3. Third generation computer
1965-1975
Integrated circuits
Further reduction in size and cost, increased reliability
First minicomputers
Time-shared operating systems
Appearance of the software industry
First set of computing standards for compatibility between systems
4. Fourth generation computer
1975-1985
Large-scale and very-large-scale integrated circuits
Further reduction in size and cost, increased reliability
First microcomputers
Growth of new types of software and of the software industry
Computer networks
Graphical user interfaces
5. Fifth generation computer
1985-present
Ultra-large-scale integrated circuits
Supercomputers and parallel processors
Laptops, tablets, smartphones, and handheld wireless devices
Mobile computing
Massive external data storage devices
Ubiquitous computing
High-resolution graphics, visualization, virtual reality
Worldwide networks and cloud computing
Miltimedea user interface
Widespread use of digitized sound, images, and movies

Fortran & Cobal

Fortran
1. What does the acronym FORTRAN stand for?
Formula Translation
2. When was FORTRAN developed?
1957
3. What was it used for?
Shortened the process of programming by using natural-language and was designed for numeric and scientific computing.
4. Is FORTRAN still being used today? If so, what for?
Yes, it is still being used due to its affinity for arithmatic. Many people in science and industrial programming favor having this program in their belt.
COBAL
1. What does the acronym COBAL stand for?
Common Business-Oriented Language
2. When was COBAL developed?
1959
3. What was it used for?
It was a english-like program designed for business uses
4. Is COBAL still being used today? If so, what for?
Yes, thogh many new programmers are not learning it. It is well imbeded in our banking systems, traffic systems, and many public-facing infrastructures in general.
